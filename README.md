# 🎮 Steam Workshop Downloader

> *"Потому что подписываться в Steam — это слишком просто"*

Привет! Это маленькая утилита, которая родилась из желания качать моды для **People Playground**, а выросла в универсальный загрузчик для любого **Steam Workshop**. Просто вводишь AppID игры и ID мода — и он твой. Никакой магии, только SteamCMD под капотом.

---

## ✨ Что умеет

- 🔍 **Находить моды** по ID из любого Steam Workshop
- 🖼️ **Показывать превьюшку** прямо в интерфейсе (да, картинку!)
- 📥 **Скачивать моды** в удобную папку (хоть по ID, хоть по названию)
- 🎯 **Работает с любой игрой**: People Playground, Garry's Mod, RimWorld, Cities: Skylines, что угодно
- 📝 **Вести подробный лог** всего происходящего
- 🚀 **Запускаться как обычная программа** (скомпилировано в .exe)
- 🌐 **Открывать страницы** мода и автора в Steam
- 📁 **Быстрый доступ** к папке с загрузками и лог-файлу

## 🛠️ Как запустить

### 🟢 Если ты хочешь просто пользоваться

1. Скачай `SteamWorkshopDownloader.exe` из Releases (релизов)
2. Запусти (Windows может ругнуться, но всё безопасно)

## 🎯 Как пользоваться

| Шаг | Действие |
|-----|----------|
| 1 | Введи **AppID игры** (для People Playground это `1118200`, но можно любой) |
| 2 | Введи **ID мода** из Steam Workshop (циферки в конце ссылки) |
| 3 | Нажми **«Найти»** |
| 4 | Полюбуйся на превью и информацию о моде |
| 5 | Нажми **«Скачать мод»** |
| 6 | Выбери, как назвать папку: по ID или по названию мода |
| 7 | Готово! Мод лежит в папке `workshop_downloads` |

## 🐛 Что-то пошло не так?

| Проблема | Решение |
|----------|---------|
| **Мод не скачивается** | Проверь `steamcmd.log`, там всё подробно. Возможно, мод требует владения игрой |
| **SteamCMD ругается** | Некоторые моды недоступны анонимно. Попробуй подписаться вручную в Steam |
| **Превью не грузится** | Steam иногда отдаёт битые ссылки на картинки |
| **Антивирус ругается** | Это ложное срабатывание, программа безопасна |
| **Не запускается** | Убедись, что установлен Python 3.8+ и все зависимости |

## 🎨 Немного истории

Вообще это начиналось как *«хочу скачивать моды для People Playground без запуска Steam»*. Потом я подумал *«а почему только для PPG?»*. Потом *«а давайте добавим превьюшки»*. А потом ещё *«а давайте нормальный GUI»*. И вот мы здесь.
Изначально программа была консольной, потом обзавелась интерфейсом на Tkinter, пережила несколько редизайнов и превратилась в универсальный инструмент.

## ⚠️ Дисклеймер

Эта штука использует **SteamCMD** (официальный инструмент Valve) и **Steam Web API**. Она не нарушает правил, но имей в виду:

- Анонимный вход работает для большинства модов
- Некоторые моды требуют владения игрой
- У Steam есть лимиты на запросы — не спамь
- Программа не позволяет обходить авторские права или платный контент
